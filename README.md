# Gift Grabber

## Introduction

A desktop and mobile browser game based on <a href="http://hol.abime.net/5578">Flag Catcher</a> for the Commodore Amiga. Created to learn Bootstrap, Backbone.js (plus Marionette) and Animate.css.
